# MIL_microestructura_CR
Code and reproducible analysis supporting the study of microstructural determinants of interest rate formation in the Costa Rican Integrated Liquidity Market (MIL). The repository provides an HTML document allowing full replication of the empirical results.

## Reproducibility
All results in the paper can be reproduced by opening the HTML file provided in this repository. The document includes all data processing steps, estimation procedures, and figures.
